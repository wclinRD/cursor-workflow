# Cursor Todo — 任務追蹤模板

> 使用方式：複製此檔案到 `.cursor/todo.md`，Agent 會自動更新

---

## 📋 使用說明

### 狀態定義

| 狀態 | 圖示 | 意義 |
|------|------|------|
| pending | ⬜ | 尚未開始 |
| in_progress | 🔄 | 進行中（同時只有一個） |
| completed | ✅ | 已完成且驗證 |
| blocked | ❌ | 被阻塞，需處理 |
| cancelled | 🚫 | 已取消 |

### 更新規則

1. **一次一項 in_progress**：同時只有一個任務在進行
2. **完成才標記**：確認驗證後才標記 completed
3. **即時更新**：每完成一項就更新此檔案
4. **blocked 要記錄**：標記 blocked 時必須說明原因
